モンダイラボ 仕様書


============================================================
概要
============================================================

アプリ名:   モンダイラボ
ジャンル:   AIが出題する試験対策アプリ
名前の由来: AIが問題を生み出す「研究所」

アプリの思想
  資格試験や検定の参考書は、分厚くて文字だらけで、とっつきづらい。
  「何から始めればいいかわからない」「読んでも頭に入らない」人が多い。
  モンダイラボは、そんな難しい教材をAIの力でポップでわかりやすい
  クイズ形式に変換するアプリ。
  堅苦しい勉強をゲーム感覚の体験に変える。

デザインの方向性
  明るくカラフルな配色 (堅い勉強アプリ感を排除)
  正解/不正解の演出を派手に (紙吹雪、アニメーションなど)
  絵文字やアイコンを多用して視覚的にわかりやすく
  AI解説は教科書口調ではなく、フレンドリーで噛み砕いた文体
  進捗表示をゲームのレベルアップのように演出
  難しい用語にはAIが「ひとことメモ」を自動付与

どんなアプリ?
  教科書や資料の内容を入力すると、AIが試験問題を自動生成してくれる学習ツール。
  選択式/穴埋め/記述式の問題を作成し、解答後にAIがわかりやすく採点/解説する。
  AIの解説は「なぜそうなるか」を日常の例えを使って説明する。

誰が使う?
  資格試験に挑戦したいけど参考書が難しくて挫折した人。
  試験前に効率よく対策したい学生。
  堅い教材が苦手で、もっと楽しく勉強したい人。


============================================================
機能一覧
============================================================

教材登録
  テキストエリアに教科書の内容やノートをペースト
  科目名 + 単元名を付けて保存
  複数の教材を科目別に管理
  教材の削除

AI問題生成
  登録した教材からGemini APIが自動で問題を作成
  問題形式を選択可能
    選択式 (4択) / 穴埋め式 / 一問一答 (記述式)
  難易度設定 (やさしい / ふつう / むずかしい)
  出題数を指定 (5問 / 10問 / 20問)

解答と採点
  1問ずつ解答していくクイズ形式
  正解で紙吹雪エフェクト + 称賛メッセージ
  不正解で優しい励ましメッセージ
  AIの解説は日常の例えを使ってフレンドリーに
  記述式はAIがキーワードベースで採点

テスト結果
  スコアをゲーム風に表示 (絵文字 + メッセージ)
  パーフェクト/すばらしい/いいぞ/これからの4段階メッセージ
  間違えた問題の一覧表示

学習履歴
  各テストのスコアを記録
  総問題数と平均正答率
  過去のテスト結果一覧 (科目、形式、日付、スコア)


============================================================
画面構成
============================================================

ホーム画面
  ヒーローバナー (グラデーション、ポップなメッセージ)
  クイック統計 (連続日数 / 正解数 / 科目数)
  科目カード一覧 (タップでクイズ設定画面へ)
  教材未登録時は登録誘導ボタン

教材管理画面
  科目名 / 単元名 / テキストの入力フォーム
  保存ボタン
  登録済み教材リスト + 削除ボタン

クイズ設定画面 (モーダル)
  問題形式ピル (4択 / 穴埋め / 一問一答)
  難易度ピル (やさしい / ふつう / むずかしい)
  出題数ピル (5問 / 10問 / 20問)
  スタートボタン

テスト画面 (全画面)
  プログレスバー + 問題番号表示
  問題文 + 回答エリア
  中断ボタン
  ローディング表示 (AI問題生成中)

フィードバックパネル (画面下からスライド)
  正解/不正解アイコン + AI解説テキスト
  次の問題/結果を見るボタン

結果画面 (全画面)
  結果絵文字 (バウンスアニメーション)
  スコア表示 + 正答率 + 応援メッセージ
  間違えた問題の確認ボタン

履歴画面
  統計カード (総問題数 / 平均正答率)
  テスト結果リスト

設定画面
  Gemini APIキー入力フィールド + 保存ボタン
  AI Studio取得リンク


============================================================
技術構成
============================================================

フレームワーク: なし (HTML / CSS / JavaScript)
データ永続化:  LocalStorage
AI:            Gemini API (問題生成、採点、解説)
エフェクト:    Canvas (紙吹雪)、CSS Animations
グラフ:        Canvas API

ファイル構成
  mondai-lab/
    docs/
      仕様書.txt       この仕様書
    index.html          メインHTML
    style.css           スタイルシート
    app.js              アプリロジック
    ai-config.js        Gemini API管理


============================================================
LocalStorageキー
============================================================

mondai-lab-materials    教材データの配列
                       各教材: subject, unit, text, createdAt
mondai-lab-history      テスト結果の配列
                       各結果: subject, format, difficulty, total, correct, accuracy, date
gemini-api-config       APIキー (ai-config.js 共通)


============================================================
AI連携設計
============================================================

Gemini APIモデル: gemini-2.0-flash
APIキーはユーザーが設定画面から入力、LocalStorageに保存

プロンプト設計
  教材テキストをコンテキストとして渡す
  問題形式/難易度/出題数を指定
  JSON形式で出力を要求 (jsonMode: true)
  解説は「中学生でもわかるように」「日常の例えを使って」と指示
  フレンドリーな口調 (「〜だよ」「〜だね」) を使用するよう指示
