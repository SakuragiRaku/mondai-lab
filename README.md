# 🧪 モンダイラボ

AIが出題する試験対策アプリ。

教科書や参考書の内容を入力すると、AIがポップでわかりやすいクイズを自動生成。堅苦しい勉強をゲーム感覚の体験に変えます。

## 機能

- 📚 教材登録 (科目・単元ごとに管理)
- 🤖 AI問題生成 (4択・穴埋め・記述式)
- 🎯 3段階の難易度 (やさしい・ふつう・むずかしい)
- 🎉 正解で紙吹雪エフェクト
- 💡 AIによるフレンドリーな解説
- 📈 学習履歴と正答率の記録

## 使い方

1. `index.html` をブラウザで開く
2. ⚙️ 設定から Gemini API キーを設定
3. 教材を登録してクイズに挑戦！

## 技術

HTML / CSS / JavaScript / Gemini API / LocalStorage
